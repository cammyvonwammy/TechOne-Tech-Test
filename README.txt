## Setup Instructions
1. Clone the repository: `git clone <repository-url>`
2. Navigate to the project directory: `cd PathTo\Cameron-Brett-TechOne-Technical-Test`
3. Install dependencies: 
    - dotnet restore
    - npm install (may need to be in test project directory)
4. Start the application: `dotnet run` (in WebAppProject directory)

## Testing
When in WebAppProject.Test directory
- Run unit tests: `dotnet test`
- Run end-to-end tests with Cypress: `npx cypress run`
